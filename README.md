# 📚 我的书架

在线 PDF 电子书阅读器，液态玻璃 UI，由 GitHub Pages 托管。

## 添加新书（仅管理员）

1. 将 PDF 文件放入 `books/` 文件夹并提交
2. 在 `books.json` 中新增一条记录：

```json
{
  "id": "唯一ID（数字或字母）",
  "name": "书名",
  "author": "作者",
  "cover_color": ["#起始色", "#结束色"],
  "file": "books/你的文件名.pdf",
  "description": "一句话简介"
}
```

3. 推送到 `main` 分支，GitHub Pages 自动更新

## 访客功能

- ✅ 浏览书架
- ✅ 在线阅读 PDF
- ✅ 自动保存阅读进度（浏览器本地）
- ✅ 翻页、缩放、全屏、跳页
- ❌ 无法上传书籍

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `→` / `空格` | 下一页 |
| `←` | 上一页 |
| `+` | 放大 |
| `-` | 缩小 |
| `F` | 全屏 |

## 部署

1. 仓库 → Settings → Pages → 选 `main` 分支
2. 访问 `https://3281341052g-bot.github.io/dian/`
